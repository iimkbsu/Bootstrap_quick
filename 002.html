grid system layout



부트스트랩 그리드 레이아웃

   부트스트랩은 기본적으로 12개의 column으로 한 개의 웹 페이지를 구성함.
   div class container로 전체 컨탠츠를 감싼 후, div class 4개의 row안에 각각 3개의 div class col로 구성 (4*3 구성)
   보통 sm, md, lg를 많이 씀.

   md의 경우 768px 이하로 좌우 페이지가 줄어들면 col이 아래로 붙으면서 모바일에서 보기 좋게 바뀜.
